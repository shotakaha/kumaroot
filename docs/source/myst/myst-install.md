# インストールしたい

```console
$ npm install -g mystmd

$ myst --version
v1.1.12

$ which myst
/opt/homebrew/bin/myst
```

``npm``を使って``mystmd``をインストールします。
とりあえずシステム全体にインストールしてOKです。

:::{note}

v1がリリースされたときに``myst-cli``から``mystmd``に名前が変わりました。

:::

## アップデートしたい

```console
$ npm update -g myst-cli
```
