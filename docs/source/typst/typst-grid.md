# グリッドしたい（`grid`）

```typst
#grid(
  columns: 3,
  rows: 4,
  gutter: 1em,
  align: (center, center, center),
  // stroke: 1pt,
  [グリッド1], [グリッド2], [グリッド3],
  [グリッド4], [グリッド5], [グリッド6],
  [グリッド7], [グリッド8], [グリッド9],
  [グリッド10], [グリッド11], [グリッド12],
)
```

`grid`関数で、行（`rows`）と列（`columns`）を指定してグリッドレイアウトを作成できます。
割付印刷したいときや、複数の図版を整列させたいときなどに便利です。

:::{note}

グリッドは、見た目上は表組み（や段組）のように見えますが、あくまでレイアウトのための要素であり、表組みや段組とは異なります。

表にするか、グリッドにするかは、文章構造を考えて、使い分けてください。

:::
