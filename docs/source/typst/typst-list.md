## 説明リストしたい（``#terms``）

```text
/ 単語: 説明
/ りんご: あか
/ ばなな: きいろ
/ ぶどう: むらさき
/ 複数行: 説明は改行しても
 先頭にスペースをいれたら
 ひとつづきにできるよ

#set terms(オプション)
#terms.item[単語][説明]
#terms.item[りんご][あか]
#terms.item[ばなな][きいろ]
#terms.item[ぶどう][むらさき]
#terms.item[複数行][説明は改行しても先頭にスペースをいれたらひとつづきにできるよ]
```
