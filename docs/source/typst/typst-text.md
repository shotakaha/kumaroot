# 和文したい（``#text``）

```typst
#set text(
    font: (
        "HackGen",
        "Noto Sans CJK JP",
    ),
    size: 12pt,
    weight: "regular",
    lang: "ja"

)
```

日本語のドキュメント作成のときにコピペで使い回している設定です。
ドキュメントのはじめに``#set text``することで一括設定しています。
フォントは複数設定でき、利用可能なフォントが見つかるまで探してくれます。
