==================================================
リモートリポジトリの設定
==================================================

後述の :command:`git clone` をすると、
リモートリポジトリは自動で設定されるので、
実はあんまり使う機会がないかも。

そのため、使い方を忘れやすいので、ヘルプで確認するとよい。

.. code-block:: bash

   $ git remote -h

.. code-block:: bash

   $ git remote -v
