# インストール

```shell
$ brew install git
$ git --version
git version 2.39.2

$ brew install git-lfs
$ git lfs --version
git-lfs/3.3.0 (GitHub; darwin arm64; go 1.19.3)
```

macOSにはデフォルトで ``git (/usr/bin/git)`` が入っていますが、Homebrewを使って最新版をインストールしておきます。
また、大きなサイズのファイルのバージョン管理することがあるかもしれないので、Git LFSも使えるようにしておきます。
