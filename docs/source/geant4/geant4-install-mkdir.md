# 作業ディレクトリを準備する（`mkdir`）

```console
// Geant4用のディレクトリを作成する
$ mkdir ~/geant4/

// ダウンロード用のディレクトリを作成する
$ mkdir ~/geant4/archives

// バージョンごとのディレクトリを作成する
$ mkdir ~/geant4/v11.X.Y
// mkdir ~/geant4/v11.2.1
```

公式ドキュメントと講習会のインストール手順を参考に、自分なりにアレンジしました。
`~/geant4/`をGeant4全般の作業ディレクトリとし、その下に
アーカイブ保存用（`archives`）と
バージョンごと（`v11.X.Y`）のディレクトリを作成します。

## ディレクトリ構成

```console
$ tree ~/geant4 -L 1
geant4
├── archives
└── v11.2.1
```

このようなディレクトリ構成になっていたら、次に進んでください。
