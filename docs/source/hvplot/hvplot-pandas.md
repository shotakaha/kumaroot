# Pandasしたい（``hvplot.pandas``）

```python
import hvplot.pandas
data.hvplot.scatter(x="X軸", y="Y軸", by="凡例")
```

``hvplot.pandas``をインポートすることで、
``pandas.DataFrame``で``hvplot``が使えるようになります。

``hvplot``は``pandas.DataFrame.plot``をモデルに拡張されたAPIとなっています。
