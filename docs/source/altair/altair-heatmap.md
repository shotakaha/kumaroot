# ヒートマップしたい

```console
chart = alt.Chart(data)
    .mark_rect()
    .encode(
        x="時間",
        y="曜日",
        color="ページビュー"
    )
```

アクセスログのヒートマップを作成する場合を想定しています。
あらかじめデータフレームを整理して``時間``、``曜日``、``ページビュー``を計算しておきます。
アクセス日時が分かっていれば、時間アクセサーを使って時間（``.dt.hour``）と曜日（``.dt.weekday`` / ``.dt.day_name``）を取得できます。
