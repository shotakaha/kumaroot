# コンテナを確認したい（`compose ls`）

```console
// Composeの状態を確認
$ docker compose ls
```

`docker compose ls`で、`compose.yaml`で管理しているたコンテナの一覧を表示できます。
コンテナの状態はDocker Desktopからも確認できます。

:::{note}

Docker Compose v2以降で利用できるコマンドです。

:::
