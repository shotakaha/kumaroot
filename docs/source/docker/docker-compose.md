# docker composeしたい

Docker Composeで複数のコンテナーを管理する基本的な流れです。
設定ファイルの作成から、コンテナーの起動・確認・操作、そして停止まで、実際に使う順番に並べてみました。

```{toctree}
---
maxdepth: 1
---
docker-compose-yaml
docker-compose-up
docker-compose-ps
docker-compose-logs
docker-compose-exec
docker-compose-run
docker-compose-cp
docker-compose-ls
docker-compose-down
```
