# 標準偏差したい（``pandas.DataFrame.std``）

## 標準偏差

:::{math}

s = \sqrt{ \frac{1}{n} \sum_{i=1}^{n} (x_{i} - m)^{2} }

:::

## 不偏標準偏差

:::{math}

s = \sqrt{ \frac{1}{n-1} \sum_{i=1}^{n} (x_{i} - m)^{2} }

:::
