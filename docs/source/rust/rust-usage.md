# Rustの使い方

最近流行りの``Rust``を学んでみようと思います。
[Tour of Rust](https://tourofrust.com/)という学習用サイトがあるので、まずはそこで学んだことを整理してみます。

```bash
$ brew install rust
$ rustc --version
rustc 1.64.0
$ cargo --version
cargo 1.64.0 (387270bc7 2022-09-16)

$ brew install rustup-init
$ rustup-init --version
rustup-init 1.25.1 (2022-07-12)

```
