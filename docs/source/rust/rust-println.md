# 標準出力したい（``println!``）

```rust
fn main() {
    println!("こんにちは、🦀");
}
```

標準出力するには``println!``マクロを使います。
関数名の末尾に``!``が必要です。
RustはデフォルトでUnicode文字列をサポートしているので、日本語も遠慮せず使えます。

## 変数を表示したい

```rust
fn main() {
    let x = 42;
    println!("生命、宇宙、そして万物についての究極の疑問の答えは{}です", x);
}
```

``{}``を使うと変数を代入して表示できます。

## 配列を表示したい

```rust
fn main() {
    let nums: [i32, 4] = [1, 2, 3, 4];
    println!("全部 : {:?}", nums);
    println!("一部 : {}", nums[2]);
}
```

配列は``0``から開始です。
