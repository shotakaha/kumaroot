# 型チェックしたい（`pyright`）

```console
$ pyright ファイル名 or ディレクトリ名
```

## インストールしたい（`pyright`）

- `pipx`でインストール

```console
$ pipx install pyright
```

- `poetry`でインストール

```console
$ poetry add pyright --dev test
```

- `uv`でインストール

```console
$ uv tool install pyright
```
