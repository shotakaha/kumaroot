# ステージ／アンステージ

``Untracked files``の{file}`source/emacs-magit.rst`にカーソルを当て
{kbd}`s`を押すとファイルがステージできます
（{numref}`fig-magit-stage`）。

ステージされたファイルは ``Staged changes`` に移動します。

ファイルを間違えてステージしてしまったなど
アンステージしたい場合は、そのファイルの先頭で{kbd}`u`を押します。
