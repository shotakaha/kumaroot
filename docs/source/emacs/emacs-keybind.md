## 検索コマンド

-  Emacsの場合、[cmigemo](https://github.com/koron/cmigemo)と[migemoパッケージ](https://github.com/emacs-jp/migemo)を導入するとローマ字で日本語検索が可能になります。
-  インストールと設定の詳細は[るびきち「日刊Emacs」](http://emacs.rubikitch.com/migemo/)を参考にするとよいと思います。
