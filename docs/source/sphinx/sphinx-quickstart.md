==================================================
ドキュメントの新規作成
==================================================

ドキュメントの新規作成には :command:`sphinx-quickstart` を使います。
既存のプロジェクト内にドキュメントを新規作成する場合、
:file:`MyPROJECT/docs/` という Sphinxドキュメント用ディレクトリを
作成するのがよいと思います。


.. code-block:: bash

   $ cd MyPROJECT
   $ mkdir docs
   $ cd docs
   $ sphinx-quickstart
