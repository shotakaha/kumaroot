# コメントしたい（``%``）

```md
% コメントの内容

% 複数行のコメント
% 複数行のコメント
```

MySTではコメントしたい場合は、行頭に``%``をつけます。

:::{seealso}

reST形式で書くと次のようになります。

```rst
.. コメント

.. 複数行のコメント
   複数行のコメント

   これも複数行のコメント
```

コメントしたい行の先頭に``.. ``をつけます。
複数行のコメントをしたい場合は、行頭を揃えます。

:::
