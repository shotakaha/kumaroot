==================================================
表紙の設定（ ``latex_logo`` ）
==================================================

表紙が寂しい場合、ロゴを挿入することもできます。
必要ないなら ``None`` （デフォルト値）にしておけばよいです。

.. code-block:: python

    # The name of an image file (relative to this directory)
    # to place at the top of the title page.
    latex_logo = './images/toumin_kuma.png'
