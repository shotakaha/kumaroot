# NumPyの使い方

```python
import numpy as np
```

NumPy（Numerical Python）は、
Pythonで効率的な数値計算を行うための基本ライブラリです。
`import numpy as np`と読み込むのが一般的です。

NumPyは科学技術計算、データ分析、機械学習などに関連する多くのパッケージの土台となっています。
NumPyのすべての機能を理解する必要はありませんが、
ある程度使い方を把握しておくと役に立つ場合が多いです。
