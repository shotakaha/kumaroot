# YouTubeを埋め込みたい（`{{< youtube >}}`）

```go
{{< youtube 動画ID >}}
```

## オプションしたい

- `id`: 動画ID（必須）
- `title`: 動画タイトル
- `class`: class属性
- `allowFullScreen=true`
- `autoplay=false`
- `controls=true`
- `loading=eager|lazy`
- `loop=false`
- `mute=false`
- `start=秒数`
- `end=秒数`
