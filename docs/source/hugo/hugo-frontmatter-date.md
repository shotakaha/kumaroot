# 日付したい（`date`）

```toml
date = "2023-10-26T15:20:00+09:00"
publishdate = "2023-10-26T15:20:00+09:00"
lastmod = "2023-10-26T15:40:00+09:00"
expirydate = "2024-10-26T15:40:00+09:00"
```

さまざまな種類の日付を設定できます。
日付フォーマットはISO8601形式を採用し、タイムゾーン情報もつけるとよいです。
