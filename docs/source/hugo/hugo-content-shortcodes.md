# Shortcodesしたい（`{{< >}}` / `{{% %}}`）

画像の挿入やYouTubeの埋め込みに必要なHTMLタグはグループ化できます。
Shortcodes記法で、このタグのグループを、本文の任意の位置に挿入できます。

## Markdownをレンダリングしたい（`{{% %}}`）

```go
{{% ショートコード名 引数 %}}
```

```go
{{% ショートコード名 %}}
表示したい内容
Markdown記法はレンダリングされる
{{% /ショートコード名 %}}
```

Markdownを引数に含める場合は`{{% %}}`形式にします。

## HTMLしたい（`{{< >}}`）

```go
{{< ショートコード名 引数 >}}
```

```go
{{< ショートコード名 >}}
表示したい内容
HTMLタグはそのまま表示される
{{< /ショートコード名 >}}
```

HTMLタグを引数に含める場合は`{{< >}}`形式にします。
