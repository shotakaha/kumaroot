## 部分テンプレートしたい（`/layouts/partials/`）

すべてのページに共通するパーツは**[部分テンプレート機能](https://gohugo.io/templates/partials/)を使うことで、テンプレート制作にかかる手間を削減できます。

```{toctree}
hugo-partials
hugo-partials-head
hugo-partials-meta
hugo-partials-header
hugo-partials-aside
hugo-partials-breadcrumbs
hugo-ogp
```
