## 設定ファイルを変更したい（`hugo --config`）

```console
$ hugo --config 設定ファイル
$ hugo --config 設定ファイル1,設定ファイル2,設定ファイル3
```

`--config`オプションで設定ファイルのパスを変更できます。
複数の設定ファイルを指定したり、
ディレクトリを指定したりできます。
