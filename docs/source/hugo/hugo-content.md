## 設定ファイルしたい（`contentDir`）

```toml
# hugo.toml
contentDir = "content"
defaultContentLanguage = "ja"
defaultContentLanguageInSubdir = false
hasCJKLanguage = true
summaryLength = 50
enableEmoji = true
```

コンテンツに関連しそうな設定を選んでみました。
