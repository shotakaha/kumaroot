```css
p {
    width: 80vw;
    line-height: 1.5;
    margin: 1rem auto;
}
```

僕の好みの段落（``p``タグ）の設定です。
基本的に、文章が読みやすくなるようなCSSにします。

段落が詰まってしまい読みにくくならないように、行の高さを設定します。
ベースとするフォントサイズにもよりますが、``line-height: 1.5``くらいがちょうどよいと思っています。

段落と段落の間にはある程度のスペースを設けます。
``margin: 1rem``くらいがちょうどよいと思います。
