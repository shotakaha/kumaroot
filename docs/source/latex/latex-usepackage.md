# パッケージを読み込む（``\usepackage``）

```latex
% プリアンブル
\usepackage[オプション]{パッケージ名}
```

パッケージは基本的にプリアンブルに追加します。
オプションは、パッケージのドキュメントを参照してください。

## 設定の例

```latex
% プリアンブル
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage{fixltx2e}
\usepackage{otf}
\usepackage{graphicx}
\usepackage{geometry}
\usepackage{hyperref}
\usepackage{pxjahyper}
```
