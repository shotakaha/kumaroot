# UTF8を読み込みたい（``inputenc``）

```latex
% プリアンブル
\usepackage[utf8]{inputenc}
```

(u)pLaTeXで入力``.tex``ファイルのエンコーディングを指定するパッケージです。
LuaLaTeXはUTF8を前提にしているので、このパッケージは不要です。

```{note}
2018年4月から(u)pLaTeXもUTF-8が標準になっているため、この設定はもう必要ありません。
```
