==================================================
ROOTファイルを開く（ ``TFile`` ）
==================================================

``TFile`` クラスを使います。
直前の ``if`` 文の中ではファイルの存在を確認しています。
ファイルがある場合は、 ``TFile::Open`` メソッドでファイルを開いています。
ない場合は、 ``TFile::TFile`` コンストラクタで
新しい ``TFileオブジェクト`` を作成しています。
