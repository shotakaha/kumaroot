# tig

なんていうツールなんだろう？
ターミナルの中で動くGUIみたいなCUIです。

## インストール

```bash
brew install tig
```

## 使い方

```bash
tig
```

終了する場合は`q`を押します。
上下は矢印キー（{kbd}`↓` / {kbd}`↑`）の他に、vimのように{kbd}`j` / {kbd}`k`でも移動できます。
`RET`するとコミットの詳細が確認できます。

## あるブランチの履歴を確認したい

```bash
tig ブランチ名
```

## ファイル／ディレクトリの履歴を確認したい

```bash
tig ファイル1 ファイル2 ...
```
