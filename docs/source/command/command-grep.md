# grep

文字列を検索するコマンドです。
ファイルの内容を検索できます。

```bash
grep [オプション] [検索パターン] [検索するファイル ...]
```

## オプション

```bash
grep -c         # マッチした行の数を書きだす
grep -l         # マッチした行を含むファイル名を書きだす
grep -n         # マッチした行の頭に行番号をつける
grep -v         # マッチしてない行を書きだす
grep -i         # 大文字／小文字を区別せずに検索する
grep --color    # マッチした<string>の箇所を色付きで表示する
```
