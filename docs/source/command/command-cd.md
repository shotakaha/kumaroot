# ディレクトリ移動したい（``cd``）

```console
$ cd ディレクトリ名
```

ディレクトリを移動するためのコマンドです。

:::{seealso}

- [](./command-zoxide.md)

:::

## ホームディレクトリに移動したい

```console
$ cd
```

引数なしで``cd``するとホームディレクトリに移動できます。

## 親ディレクトリに移動したい

```console
$ cd ..
```

## 直前のディレクトリに移動したい

```console
$ cd -
```
