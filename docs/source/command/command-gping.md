# pingしたい（``gping``）

```console
$ brew install gping
gping 1.16.1
commit_hash:
build_time: 2024-02-17 11:44:14 +00:00
build_env: rustc 1.76.0 (07dca489a 2024-02-04) (Homebrew),
```

```console
$ gping [オプション] ホスト名|IPアドレス
```

``gping``を使うとpingした結果をグラフで表示できます。
