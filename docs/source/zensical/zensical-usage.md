# Zensicalしたい

`zensical`は`MkDocs for Material`を開発していたメンバーたちが
2025年末にリリースした新しい静的ドキュメント作成ツールです。
Markdownで書かれたファイルを見やすいウェブページとしてまとめることができます。

MkDocsおよびMkDocs for Materialの開発で得られた知見を活かしつつ、
ゼロから再構築したツールですが、`MkDocs for Material`の設定はほとんどそのまま使えるそうです。

また、MkDocs 2.0では、これまでのテーマの仕組みが廃止される予定であることがアナウンスされています。
このままでは、`MkDocs for Material`そのものも使えなくなるため、
`zensical`への移行を検討しています。

zensical-readthedocs
